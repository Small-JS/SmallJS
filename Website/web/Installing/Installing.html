Installing SmallJS

The full, free source code of SmallJS can be downloaded from GitHub here:

github.com/small-js/smalljs

Just download and extract the zip file under the code button or clone the repository with a Git client. You could use GitHub Desktop if you're new to Git.

The file README.md on the main page shows how to install the necessary prerequisites like Visual Studio Code, Node.js and Git (bash) if you run Windows.
