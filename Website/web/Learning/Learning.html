Learning Smalltalk

If you're new to the wonderfully elegant Smalltalk language, it's probably a good idea to study one or more of these resources.

It's best to first try these tutorials in the Smalltalk implementations they are written for, because in SmallJS, the names of classes and methods are based on their JS equivalents, not on the Smalltalk-80 standard.

You can then also appreciate the different approach of SmallJS compared to image-based Smalltalk implementations with and integrated IDE.
Resource Description
Learn Smalltalk with professor Stef Brief tutorial in Amber Smalltalk.
Learn X in Y minutes Where X = Smalltalk
Squeak by example Full book based on Squeak Smalltalk.
Digitalk Smalltalk/V Tutorial and Programming Handbook You can't run Smalltalk/V anymore,
but chapters 1 and 3 are excellent into's to Smalltalk concepts.
MDN DOM HTML Modzilla Developer Network
SmallJS class and method names match mostly to this.

Syntax of SmallJS

The SmallJS syntax is mostly compatible with Smalltalk-80
Some keywords have been added to allow it to be file based and call JS functions.

A quick guide to the SmallJS syntax and its source code organisation
can be found in this subfolder on GitHub: Smalltalk/Smalltalk.md
